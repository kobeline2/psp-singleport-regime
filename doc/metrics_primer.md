## 0. 全体の設計 — 3つの層 + 1本の物差し + 1つの舞台

```
層1  表面速度場  u(x,y,t)      ← PIV が測る / CFD が出す (毎フレーム数千〜1.5万ベクトル)
        ↓ s30 (フレームごとに 1 個のスカラーへ圧縮)
層2  フレーム指標  E(t), phi_lv(t), I_asym(t), I_circ(t), I_rot(t)
        ↓ s31 (各フレームに水深タグ h(t) を貼る) → s32 (バンドで集計)
層3  バンド指標  E_mean, ..., I_unst(B_k), E_snr     ← 論文の表・図になる層
```

- **物差し** `U_p = Q / (a b)`: ポート断面平均速度 (a=0.055 m, b=0.11 m).
  Q は**その流量水準の公称値** (metadata/flow_levels.csv) を使う — 実測 Q は走行中に
  変動するため, 条件間比較の土俵を固定する目的で公称に統一している.
  例: medium (0.50 L/s) → U_p = 0.0826 m/s.
- **舞台** `Ω`: 解析領域. 実験ではポート近傍を除外したマスク領域
  (反射・歪み・三次元性のため. マスクは全条件共通). すべての ⟨·⟩ は Ω 上の面平均.

## 1. 各指標カード

### E(t) — 「表面がどれだけ動いているか」

```
E(t) = ⟨ |u|² ⟩_Ω        [m²/s²]
```

- 直感: 表面運動エネルギーの面平均 (質量を除いた). E = 0.001 なら典型速度 ~3.2 cm/s.
- **次元を持つ唯一の主指標** → 流量が違うラン同士は生の E を比べてはいけない.
  必ず `E / U_p²` で無次元化してから比較する.
- 安全装置: PIV ノイズ床の実測値 `E_noise_floor = 2e-7 m²/s²` に対して
  SNR = E_mean / E_noise ≥ 3 を満たさないバンドは `E_resolved = false` が立ち,
  その帯の E 系指標 (E, I_unst, I_circ, I_rot) は定量に使わない.

### phi_lv(t) — 「表面のどれだけが死んでいるか」

```
phi_lv(t) = Area(|u| < u_th) / Area(Ω),      u_th = 0.2 · U_p
```

- 直感: 低速域の面積率. E の鏡像だが**速度分布の形に敏感** —
  同じ E でも「細いジェット + 広い死水域」なら phi 大, 「全体がだらだら動く」なら phi 小.
  E と phi を並べて初めて「エネルギーの配り方」が見える.
- 閾値は**相対** (0.2 U_p). α = 0.2 は「inflow で飽和せず h/a 依存を分解できる」基準で
  パイロット解析から固定 (metrics policy: 以後動かさない).
- outflow で phi ≈ 1 に飽和するのは指標の不具合ではなく
  「port の駆動が自由表面までほとんど届かない」という物理所見として扱う (methods.tex).

### I_asym(t) — 「左右どちらかに偏っているか」

```
I_asym(t) = |E_L − E_R| / (E_L + E_R)        (実装は絶対値)
```

- 直感: 左右半分のエネルギー配分の偏り. 0 = 対称, 1 = 全部片側.
- **絶対値である点が重要**: 蛇行する流れ (偏りが右↔左を行き来する) は,
  符号つき平均なら ≈0 に消えるが, 絶対値平均なら「瞬間ごとの偏りの強さ」として残る.
  つまり band-mean の I_asym は「偏向がロックしているか」ではなく
  「瞬間的にどれだけ偏っているか」を測る.
- E の比なので次元も U_p も消える → **流量・較正・定義バージョンに最も頑健な比較子**.
- 細部: 中線は解析領域の y 範囲の中点で切る (FOV が非対称なら物理中心とずれる).

### I_circ(t) と I_rot(t) — 「正味の回転」と「回転の活発さ」

```
ω = ∂v/∂x − ∂u/∂y                         (2次元渦度)
I_circ(t) = (a/U_p) · ⟨ω⟩_Ω               (符号つき: 正味どちら回りか)
I_rot(t)  = (a/U_p) · ⟨|ω|⟩_Ω             (符号なし: どれだけ回っているか)
```

- **なぜ 2 つ要るか**: ⟨ω⟩ ≈ 0 になるのは「回転が無い」ときと
  「双子渦が打ち消し合う」ときの両方. I_rot はこの 2 つを区別し,
  回転運動が実在するときだけ大きくなる.
- 正規化 (a/U_p) により Q 依存がほぼ消える → 流量間でそのまま比較可.
- 渦度は微分量なのでノイズに弱い. 平滑 `smooth_sigma_circ` は現在 0 (平滑なし).
  変更するなら全ラン共通で適用すること.
- 符号規約: piv.y が増える向きに依存する (実装ヘッダの注意書きを参照).

### I_unst(B_k) — 「その水深帯で E がどれだけ暴れたか」(唯一のバンド専属指標)

```
I_unst(B_k) = std{E(t) | h(t)∈B_k} / mean{E(t) | h(t)∈B_k}      (変動係数)
```

- 直感: 帯に滞在している間の E の相対的な揺れ. レジームの「不安定さ」の代理.
- 比なので次元・流量フリー. ただし **2 つの交絡**に注意:
  1. **トレンド交絡**: 帯の中で E が単調に成長するだけでも std が立つ
     (「揺らぎ」ではなくても大きく出る). 解釈時は E(t) の時系列を必ず目視すること.
  2. **推定分散**: フレーム数が少ないと std/mean の推定が粗い.
     実験はバンドあたり 10³ 級, CFD (5s サンプリング) は 10¹ 級 — 比較時は留意.

## 1b. 深掘り: 「回っている」を正しく測る — I_circ vs I_rot と組織化度

渦度 ω は**その場の流体粒子の自転**(局所)である. 一方「水槽全体がどちら回りか」は**大域の循環**で,
両者は別物 — 局所の自転が激しくても向きが揃わなければ大域の渦にはならない (双子渦は打ち消し合う).
この区別を測るのが I_circ と I_rot の対だった (§1 のカード参照). ここではその発展を扱う.

### 組織化度 = |I_circ| / I_rot

```
組織化度 = |<ω>| / <|ω|>       0〜1
  1 に近い → 自転が全部同じ向き = 単一の大渦にまとまっている
  0 に近い → 自転が左右で打ち消し合う = まとまらない対称流
```

「回転活性 (I_rot) はあるのに, 大域の渦 (I_circ) になっていない」状態を 1 つの数で捕まえる.
`s33_compare_cfd_experiment.m` が `I_circ_org` として出す. 実例 (2026-07): CFD は I_rot が高い
(局所回転は多い) のに組織化度 ~0.005 で, 単一渦にまとまらない — E だけ見ていたら見えない差だった.

### 循環の脈動 = I_circ_std / |I_circ_mean|

E に対する I_unst と同じ発想で, 循環そのものの時間変動係数. 大循環が定常か脈動かを測る.

### 符号つき I_circ(t) は「対称性の破れ」を暴く — 最重要の落とし穴

band_metrics の I_asym は**絶対値** (§1 の I_asym カード) なので, 左右どちらに偏っているかは消えている.
偏りの**向き**を知るには**符号つき I_circ(t)** を見る. これが決定的な場面がある:

> 単一ポート inflow は自発的に左右対称性を破る (ジェットがどちらかの壁に付着してロック).
> 同じ条件の実験反復でも, ある run は時計回り (I_circ<0), 別の run は反時計回り (I_circ>0) に
> 固定されうる (双安定). 実測 (2026-07, medium): R0006 は I_circ 平均 -0.0078 で常に負,
> R0008 は +0.0077 で常に正 — **向きが真逆の同一条件反復**. これが対称性の破れの指紋.

I_circ(t) の3つの読み筋:

```
|平均|/std が大 (>~5), 符号反転ほぼ 0%  → 定常な単一渦, 片回りに固定 (対称性が破れてロック)
|平均|/std が小, 符号反転が多い          → 蛇行 (向きが時間で入れ替わる) or 対称で未決
正の割合が 0% か 100% 付近               → 片側にロック / 50% 付近 → 左右拮抗 (破れていない)
```

### 比較の鉄則: 別ブランチの反復で符号つき量を平均するな

対称性が破れる系では, 反復が別ブランチ (R0006=CW と R0008=CCW) に落ちる. このとき
**符号つき I_circ を反復平均すると打ち消して ~0 になり無意味**. 比較・集計に使うのは
**符号不変量**に限る: |I_asym|, |I_circ| (magnitude), 組織化度, I_unst. 各反復がどのブランチかは
符号つき I_circ で個別に見て, その上で「どちらのブランチも実現する双安定な系」として扱う.

### 渦度で渦の中心を探すな (検出の落とし穴)

「渦の中心 = 渦度 |ω| の最大点」は素朴だが誤り. コヒーレントなジェットが走ると, 最強の渦度は
渦核ではなく**ジェット両脇の剪断層**に出る (§1 の自由渦の話と同根: 回転構造と渦度の大小は別).
渦核の位置を追うなら **Q 判定法 / λ2 法** (回転が歪みを上回る場所) か**流れ関数の極値**を使う.

## 2. 正規化依存マップ — 「何と何を比べてよいか」

| 指標 | 次元 | Q への感度 | 流量違いの比較 | 注意点 |
|---|---|---|---|---|
| E | m²/s² | **∝ Q² (強)** | `E/U_p²` 必須 | 生の E を跨いで比べない |
| phi_lv | 無次元 | 中 (u_th ∝ U_p 経由) | 同じ α なら可 | 定義 (相対閾値) の版に注意 |
| I_asym | 無次元 | ほぼ無し | そのまま可 | abs である意味 / 中線 / Ω |
| I_circ, I_rot | 無次元 | 弱 (a/U_p 済み) | そのまま可 | 微分量ゆえ解像度・平滑に注意 |
| I_unst | 無次元 | 無し | そのまま可 | トレンドとフレーム数に敏感 |

覚え方: **E だけが「生の量」で, 残りは全部「比」**. 右に行くほど比較は頑健.

さらに実務上の注意 — 「公称 Q」と「実測 Q」:
ポンプの実流量は走行中に変動しうる (バンド透過時間や q_actual 列で確認できる).
E/U_p² を**公称**で作るのは条件ラベル間の土俵合わせ, **実測**で作るのは物理の
スケーリング検証 — 目的が違う 2 つの正規化であり, どちらを使ったかを明記すること.

## 3. 実データを 1 行, 声に出して読む

R0006 (inflow, medium) の B1 行 (band_metrics.csv):

```
E=0.00110  phi_lv=0.332  I_asym=0.354  I_rot=0.097  I_unst=0.408
```

> 「浅水帯では表面が典型 3.3 cm/s で動き (E), それでも面積の 1/3 は閾値以下で淀む (phi).
> エネルギーは瞬間ごとに左右どちらかへ強く偏り (I_asym), 回転運動もはっきり在り (I_rot),
> 帯滞在中の E は ±40% 揺れる (I_unst) — **偏向ジェットが暴れる非定常レジーム**.」

同じランの B4 は `phi 0.55, I_unst 0.05` —
「表面の半分以上が死に, 揺らぎもほぼ消えた静かな深水レジーム」.
指標 5 つで流況の人相書きができる, というのがこの体系の使い方.

## 4. CFD と比較するときのチェックリスト

1. **同じ定義の版か**: メトリクス定義を変えたコミット (例: phi_lv の相対閾値化) を跨いだ
   derived データ同士を比べていないか. band_metrics の生成日時とコミットを照合する.
2. **同じ正規化か**: E は E/U_p² に直したか. 公称 Q か実測 Q かを揃えたか.
3. **同じ Ω か**: 実験の解析領域 Ω = PIVグリッドの範囲そのもの
   (x∈[0.045, 2.909], y∈[0.041, 1.929], 31.5mm. `C.metrics.omega_*` に定数化).
   CFD は `import_cfd_surface` に `x_lim_m/y_lim_m/grid_dx_m` でこの箱を渡して揃える.
   実測 (2026-07-24): 揃えても E は数% しか動かない (壁際の低速セル除外がポート近傍除外を
   相殺するため) — 大きな乖離はマスクの産物ではない.
4. **同じ観測演算子か**: PIV はペア間隔の平均速度・照合窓の空間平均を返す.
   CFD の瞬時・細格子の場と比べるときは, 必要に応じて同等の平滑を検討する.
5. **統計量の土俵**: フレーム数 (実験 10³ vs CFD 10¹) と実現数 (実験 n=3 反復 vs CFD n=1)
   の差を踏まえ, 「反復レンジ内か」だけでなく差の向き・形状も見る.
6. **どのブランチか**: この系は左右対称性を破る (§1b). 反復や CFD が単一渦レジームの
   どちら回りブランチにいるかを符号つき I_circ で確認し, 符号つき量は反復平均しない.
   対称メッシュの CFD は破れの種が無く対称固定点に留まりうる (= 実験のどのブランチとも
   別状態) — その場合は「不一致」ではなく「破れの再現失敗」として扱い, 摂動導入を検討.
